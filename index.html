next主题

这篇文章简要介绍下next主题的使用，具体内容请查看：next使用文档

一：安装与启用：

  1. 安装：
    $ git clone https://github.com/theme-next/hexo-theme-next
    1. 按照使用文档中的会失败，请用上面这个。
    2. 之后会得到hexo-theme-next文件夹，把它改名为next并放入themes文件夹下
  2. 启用：
    找到站点配置文件_config.yml，修改themes字段为next：
    theme: next
